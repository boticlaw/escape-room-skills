# Template: Hall Escape Básico

Estructura mínima para un juego de Hall Escape.

## Estructura de Archivos

```
game_{id}/
├── GAME.json
├── narrativa/
│   └── NARRATIVE.json
├── pruebas/
│   ├── prueba_001.json
│   ├── prueba_002.json
│   └── prueba_003.json
└── flujo/
    └── FLOW.json
```

## Configuración Mínima

- 3-5 pruebas
- Duración indicada por el usuario en el BRIEF
- Narrativa lineal simple
- 1 Game Master presente
