# Game Type: Street Escape

## Overview

Outdoor escape game played across streets, plazas, and parks. Players navigate real-world locations using GPS, QR codes, and physical exploration. Technology-heavy but keeps the escape room feel.

## Constraints

| Parameter | Value |
|---|---|
| Space | Outdoor, public streets and parks |
| Players per team | 2–5 (optimal: 3–4) |
| Duration | 45–90 minutes |
| Technology | GPS, QR codes, mobile app/tablet required |
| Staff | Zero GM (self-service via app) |

## Recommended Mechanics

| Priority | Mechanics |
|---|---|
| Core | gps-navegacion, ubicacion-qr, busqueda-objetos, exploracion-visual |
| Supporting | adivinanza-ubicacion, acrostico-ubicacion, investigacion-texto |
| Digital | laberinto-digital, arcade-digital |
| Avoid | panel-electrico, mecanismo (require fixed installations) |

## Flow Structure

- **lineal** preferred: players move from location to location in sequence
- **no_lineal** possible with hub-and-spoke design (central plaza radiating outward)
- GPS triggers location unlocks; QR codes deliver content at each stop
- Mobile device handles hint system and timer

## Design Adaptations

- All puzzles must account for weather (rain, sun, wind)
- No props that can be stolen or vandalized by public
- QR codes must be replaceable (laminated, mounted securely)
- GPS accuracy: design for 10m radius, not pinpoint
- Walking distance between locations should be 2–5 minutes maximum
- Accessibility: avoid stairs, unpaved paths, or restricted areas
- Real-world elements: integrate existing architecture, signs, monuments

## Validation Checklist

- [ ] All locations are publicly accessible (no permits needed)
- [ ] Walking distances between stops are reasonable (2–5 min)
- [ ] QR codes are weather-proof and replaceable
- [ ] GPS coordinates tested on both iOS and Android
- [ ] No puzzles depend on temporary structures or moveable objects
- [ ] Route is accessible (no stairs-only paths)
- [ ] Mobile app works offline (poor signal in some areas)
- [ ] Rain contingency: critical elements protected or have backup plan
- [ ] Players cannot accidentally skip ahead via GPS spoofing
