# Template: Street Escape Básico

Estructura mínima para un juego de Street Escape.

## Estructura de Archivos

```
game_{id}/
├── GAME.json
├── narrativa/
│   └── NARRATIVE.json
├── pruebas/
│   ├── prueba_001.json (GPS)
│   ├── prueba_002.json (QR)
│   └── prueba_003.json (Exploración)
└── flujo/
    └── FLOW.json
```

## Configuración Mínima

- 5-8 pruebas
- 60-90 minutos de duración
- Ruta con 3-5 checkpoints
- App móvil o web guide
