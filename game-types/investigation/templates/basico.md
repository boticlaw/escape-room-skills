# Template: Investigación Básica

Estructura mínima para un juego de Investigación.

## Estructura de Archivos

```
game_{id}/
├── GAME.json
├── narrativa/
│   └── NARRATIVE.json
├── pruebas/
│   ├── prueba_001.json (Evidencia)
│   ├── prueba_002.json (Deducción)
│   └── prueba_003.json (Resolución)
└── flujo/
    └── FLOW.json
```

## Configuración Mínima

- 4-6 pruebas
- 60-120 minutos de duración
- 3-5 sospechosos/personajes
- Solución lógica al misterio
