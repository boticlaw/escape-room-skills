# Game Type: Investigation

## Overview

Detective or crime-themed escape game focused on narrative, deduction, and evidence gathering. Players investigate a scene, interview suspects (via documents/audio), and piece together a solution. Can be indoor or mixed indoor/outdoor.

## Constraints

| Parameter | Value |
|---|---|
| Space | Indoor or mixed indoor/outdoor |
| Players per team | 2–6 (optimal: 3–4) |
| Duration | 60–120 minutes |
| Technology | Minimal — documents, props, optional tablet for evidence log |
| Staff | Zero GM (self-service, documents drive the narrative) |

## Recommended Mechanics

| Priority | Mechanics |
|---|---|
| Core | investigacion-texto, busqueda-objetos, exploracion-visual |
| Supporting | texto (matching), secuencial, posiciones |
| Optional | comunicacion-mensajes, tablet-cooperativo |
| Avoid | punteria-derribo, arcade-digital (break immersion) |

## Flow Structure

- **multi_lineal** with accumulation: players gather evidence from multiple sources simultaneously
- Evidence converges into a final deduction phase
- Narrative reveals through found documents, audio recordings, photographs
- Red herrings acceptable (add realism) but must be clearly resolvable

## Design Adaptations

- Heavy document design: newspapers, letters, police reports, photographs
- Each document must feel authentic (typography, aging, formatting)
- Evidence board or journal helps players track findings
- Timeline reconstruction is a common meta-puzzle
- Suspect profiles with contradictions drive the core deduction
- Player as detective: every found item should feel like a real clue
- Immersion is critical: avoid anything that breaks the fiction

## Validation Checklist

- [ ] Narrative is coherent and solvable with the provided evidence
- [ ] All documents are thematically consistent (fonts, paper, language)
- [ ] Red herrings are fair — players can eliminate them with logic
- [ ] Final deduction requires combining evidence from multiple sources
- [ ] Timeline is internally consistent (no chronological contradictions)
- [ ] Suspect profiles have clear, distinguishable traits
- [ ] Documents are legible and durable for repeated use
- [ ] Evidence accumulation is trackable (board, journal, or app)
- [ ] Solution is unambiguous (one clear answer, not multiple interpretations)
