# Nonogram Mitológico de Cupido

## ID
prueba_cupido_nonogram_001

## Skill
prueba-logica-nonogram

## Dificultad
5/10

## Descripción
Los jugadores deben resolver un nonogram (puzzle de lógica con cuadrícula) donde las pistas numéricas indican cuántas celdas consecutivas deben rellenar en cada fila y columna. Al completarlo correctamente, se revelan los nombres de figuras mitológicas relacionadas con Cupido: Cupido, Himeneo, Psique y Erotes.

## Tags
logica, nonogram, puzzle, mitologia, cupido, codigo

## Solución
Resolver nonogram 10x10 usando lógica deductiva: 1. Identificar filas/columnas con pistas definitivas. 2. Marcar celdas seguras. 3. Usar X para descartar imposibles. 4. Completar por deducción.
