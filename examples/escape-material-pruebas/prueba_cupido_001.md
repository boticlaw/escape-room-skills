# Cupido - Sincronía de Latidos

## ID
prueba_cupido_001

## Skill
prueba-tablet-cooperativo

## Dificultad
4/10

## Descripción
Juego de ritmo cooperativo donde 2-6 jugadores deben pulsar en sincronía perfecta siguiendo el latido visual y auditivo de un corazón. El sistema incluye anti-camping (obliga a levantar el dedo entre latidos), dificultad adaptativa (se ajusta por aciertos/fallos) y fail-safe (victoria automática tras 3 minutos). Al conseguir 5 aciertos consecutivos, muestra un código de victoria.

## Tags
tablet, cooperativo, ritmo, sincronizacion, latido, anti-camping, godot, android, cupido, codigo

## Solución
Sincronización de toques al ritmo del corazón: 1. Designar un 'contador' que diga 'YA' cuando el corazón se ponga rojo. 2. Levantar el dedo completamente tras cada golpe. 3. Mirar el corazón, no a los compañeros.
