# Puntería de Cupido - Derribo de Objetivos

## ID
prueba_cupido_punteria_001

## Skill
prueba-punteria-derribo

## Dificultad
4/10

## Descripción
Los jugadores usan una tirachinas con proyectiles para derribar botellas/corasones dispuestos en una estantería o mesa. Al caer, cada objetivo revela un número oculto en su base. Deben derribar todos los objetivos para obtener los números y formar el código.

## Tags
punteria, derribo, tirachinas, destreza, cupido, codigo

## Solución
Derribar objetivos con tirachinas para revelar números: 1. Recoger los proyectiles. 2. Apuntar con calma, no solo fuerza. 3. Memorizar o anotar cada número revelado.
