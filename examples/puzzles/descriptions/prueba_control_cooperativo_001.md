# Navegación Cooperativa

## ID
prueba_control_cooperativo_001

## Skill
prueba-tablet-cooperativo

## Dificultad
6/10

## Descripción
Una esfera energética debe atravesar un laberinto lleno de obstáculos. Cada jugador controla una dirección: arriba, abajo, izquierda, derecha. En hall escape usan una tablet compartida. En street escape cada uno usa su móvil y ve la misma esfera moverse en tiempo real. Deben comunicarse para guiar la esfera hasta la meta. Al completar el recorrido, reciben un código en pantalla.

## Tags
tablet, cooperativo, control, navegacion, laberinto, codigo

## Solución
Control direccional cooperativo: 1. Establecer un coordinador que llame las direcciones. 2. Mirar el laberinto completo antes de moverse. 3. Movimientos cortos y controlados. 4. Si la esfera toca un obstáculo, vuelve al inicio pero mantened la calma.
