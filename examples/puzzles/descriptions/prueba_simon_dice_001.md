# Secuencia Alienígena

## ID
prueba_simon_dice_001

## Skill
prueba-tablet-cooperativo

## Dificultad
5/10

## Descripción
Una interfaz alienígena muestra secuencias de colores que los jugadores deben memorizar y repetir. Cada jugador controla un color (rojo, azul o verde). En hall escape usan una tablet compartida. En street escape cada uno usa su móvil y ve la misma secuencia sincronizada. La secuencia se alarga hasta 7 elementos. Al completarla, aparece un código numérico en pantalla.

## Tags
tablet, cooperativo, simon_dice, memoria, secuencia, pantalla_codigo, codigo

## Solución
Simon Dice cooperativo con memoria grupal: 1. Asignar colores claros. 2. Repetir en voz alta la secuencia mientras suena. 3. Mantener el orden de turnos para pulsar. 4. Si se olvida, usar mnemotecnia (ej: 'Rojo-Azul-Rojo' = 'Raúl Baila Rap').
