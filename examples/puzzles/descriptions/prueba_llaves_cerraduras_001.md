# Las Cinco Llaves y Cerraduras

## ID
prueba_llaves_cerraduras_001

## Skill
prueba-emparejamiento-memoria

## Dificultad
5/10

## Descripción
Pepe ha conseguido 5 llaves y 5 cerraduras, cada una correspondiente a diferentes casas del pueblo. No todas las casas pueden ser el hogar que busca. El desafio es encontrar la combinacion correcta entre llaves y cerraduras, y recordar la secuencia para avanzar.

## Tags
street_escape, emparejamiento, memoria, llaves, vivienda, monzon_campos

## Solución
Emparejamiento con memoria de secuencia: Probar cada llave con cada cerradura, recordar las combinaciones, luego pulsar en secuencia
