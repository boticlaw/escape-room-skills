# CHECKLIST DE VALIDACIÓN — Protocolo Alerta Verde

## Game Design
- [x] Estructura de actos clara (4 actos: Alerta, Investigación, Contraataque, Cierre)
- [x] Flujo de puzzles documentado (6 pruebas en orden por planta)
- [x] Objetivo principal en una frase: "Detener el sabotaje de la Red Ecológica Verda"
- [x] Dificultad calibrada para público 12-18 años (rango 2-5/10)
- [x] Meta-prueba integra elementos anteriores (Código Maestro AV1P)

## Puzzle Design
- [x] Variedad de tipos: exploración visual, emparejamiento, lógica secuencial, UV, panel eléctrico, integración
- [x] Cada puzzle integrado en narrativa (justificado por la historia)
- [x] Soluciones únicas y auto-validables (candados se abren o no)
- [x] Pistas documentadas (3 niveles por prueba)
- [x] No hay más de 2 pruebas del mismo tipo

## Storytelling
- [x] Briefing < 3 minutos
- [x] Personajes con profundidad (Soler, Vega, Pardo)
- [x] Viaje emocional planificado (curiosidad → urgencia → intriga → tensión → triunfo)
- [x] 2 giros narrativos (Soler es víctima, Pardo es saboteador)
- [x] Desenlace satisfactorio (2 min de cierre)

## Psicología
- [x] Curva de dificultad progresiva (2→3→4→3→4)
- [x] Puzzles para todos los arquetipos (buscador, lógico, comunicador, creativo, líder)
- [x] Sistema de hints accesible (3 niveles, timing definido)
- [x] Mini-victorias cada ~6-8 min (6 pruebas en 50 min)
- [x] Flujo multi-lineal: 2 grupos paralelos sin interacción

## UX
- [x] Onboarding: sobre inicial con todo lo necesario
- [x] Props simples (candados, papel impreso, linterna UV)
- [x] No hay puzzles ambiguos
- [x] Montaje simple (<15 min por planta)
- [x] Materiales baratos y reutilizables
- [x] Espacio aprovechado (3 plantas, 6 habitaciones)

## Formato Competitivo
- [x] 2 grupos simultáneos con mismas pruebas
- [x] Sin interacción directa (plantas independientes)
- [x] Premio: certificado UPA
- [x] Rotación: 4 grupos por sesión (2 turnos)

## Presupuesto
- [x] Gasto contenido: papel, candados, linterna UV, cajas
- [x] Mínima decoración: carteles y documentos
- [x] Sin tecnología compleja
- [x] Materiales reutilizables para otros juegos

## Capa Educativa Ambiental
- [x] Datos reales integrados de forma natural en documentos del juego (no trivia)
- [x] Datos no bloquean puzzles — leer los documentos es necesario, memorizar datos no
- [x] Comparaciones impactantes y memorables (40 bañeras, 16.000 móviles, 9/10 personas...)
- [x] Datos locales de Castilla y León (horas de sol, bosques, lince ibérico)
- [x] Datos positivos (lince, renovables) para esperanza; datos negativos (contaminación) para indignación narrativa
- [x] Panel informativo como decoración inmersiva (no obligatorio)
- [x] 18+ datos ambientales distribuidos por las 6 pruebas
