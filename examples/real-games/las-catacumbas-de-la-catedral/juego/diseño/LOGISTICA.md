## Disposición del Espacio

### Requisitos del Espacio Joven

```
ENTRADA
  │
  ├── Recepción / Briefing (espacio abierto)
  │
  ├── SALA 1 ← Puerta con candado (GM tiene llave)
  │   └── Mesa de escritorio + corcho + lámpara ámbar
  │
  ├── SALA 2 ← Puerta con candado (GM tiene llave)
  │   └── Estantes + mesa con fragmentos + marco en pared
  │
  ├── SALA 3 ← Puerta con candado (GM tiene llave)
  │   └── 2 mesas + estantería divisoria + iluminación transición
  │
  ├── SALA 4 ← Puerta con candado (GM tiene llave)
  │   └── Mesa grande con mapa + tablero en pared + marcos con crónicas
  │
  └── SALA 5 ← Puerta con candado (GM tiene llave)
      └── Altar + velas LED + oscuro + zona tablet
```

### Dimensiones estimadas por sala
- **Sala 1**: 2×3m (escritorio + movimiento)
- **Sala 2**: 2.5×3m (estantes + mesa)
- **Sala 3**: 3×3m (dos zonas separadas)
- **Sala 4**: 3×4m (mapa grande + movimiento entre marcos)
- **Sala 5**: 3×3m (altar + zona de ensamblaje)
- **Total**: ~65m² + pasillos/conexiones

Si el Espacio Joven no tiene 5 habitaciones separadas, se pueden usar:
- Divisiones con biombos o cortinas opacas
- rincones de una sala grande con separadores
- El orden de visita es secuencial, así que se puede usar un espacio tipo "estaciones"

## Preparación de Tablets

### Tablet 1 — Intro
- **Contenido**: Audio de 30s (mensaje de la Dra. Marcos) + pantalla de espera
- **Formato**: MP3 + HTML offline
- **Carga**: Batería completa

### Tablet 2 — Ambiente (Sala 4)
- **Contenido**: Audio ambiental 30s en loop (goteo, eco)
- **Formato**: MP3 + HTML offline (autoplay)
- **Carga**: Batería completa

### Tablet 3 — Código Final (Sala 5)
- **Contenido**: Interfaz de introducción de código (VERDAD) + pantalla de revelación
- **Formato**: HTML offline + JS (sin servidor)
- **Carga**: Batería completa

### Tablet 4 — Backup
- **Contenido**: Copia de Tablet 3 (código final)
- **Formato**: HTML offline
- **Carga**: Batería completa
- **Uso**: Si Tablet 3 falla, o para grupos grandes
