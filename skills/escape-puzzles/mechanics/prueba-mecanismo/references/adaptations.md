# Adaptaciones — Mecanismo

## Por Tipo de Juego

**Hall Escape:** Mecanismos robustos (10-200 personas/dia). Panel fijo en pared/mueble. Override GM accesible pero oculto.

**Street Escape:** Version portatil (caja con asa). Proteccion climatica. Bluetooth/NFC para validar.

**Investigacion:** Formato evidencia (maleta, documentos). Objetos tematicos (llaves antiguas, candados vintage).

## Por Edad

**Ninos (6-10):** Max 3 elementos, colores muy distintos. Encajes holgados o magneticos. Sin interruptores posicionales.

**Adolescentes (11-17):** Estandar completa. Logica matematica simple.

**Adultos (18+):** Complejidad completa. Tematica profesional.

**Mixtos:** Dividir tareas (unos cables, otros interruptores).

## Sinergias

- `prueba-busqueda-objetos`: Llaves/pistas escondidas
- `prueba-cifrado`: Combinacion codificada
- `prueba-logica-secuencial`: Orden sigue logica deducible

## Alternativas

- `prueba-panel-electrico` → Si requiere cableado/circuitos
- `prueba-codigo-numerico` → Si es solo introducir codigo
- `prueba-digital-interfaz` → Si es virtual
