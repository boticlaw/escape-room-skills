# Adaptaciones — Tablet Cooperativo

## Por Edad

**Ninos (6-10):** Max 2 jugadores, ritmo lento 50-60 BPM. Animaciones divertidas. Sin tolerancias estrictas.

**Adolescentes (11-17):** Estandar completa. Elementos competitivos internos.

**Adultos (18+):** Complejidad completa. Comunicacion estrategica.

**Mixtos:** Emparejar adultos con ninos. Adultos guian verbalmente.

## Por Espacio

**Hall Escape:** UNA tablet central (min 10"), multitouch, pedestal o soporte.

**Street Escape:** Cada jugador usa su movil. Sincronizacion via servidor web (WebSocket/Firebase). Latencia objetivo <100ms. Plan B si problemas de conexion.

**Investigacion:** No aplica directamente. Alternativa asincrona con `prueba-digital-interfaz`.

**Digital/Virtual:** Servidor + videochat.

## Por Duracion

**Quick (5-10 min):** Un solo objetivo simple (5-8 aciertos). `duracion_juego_segundos=30`.

**Standard (15-30 min):** Completa. Multiples fases posibles.

**Epic (45+ min):** Multiples minijuegos consecutivos: Calentamiento → Simon Dice → Control cooperativo → Desafio final.
