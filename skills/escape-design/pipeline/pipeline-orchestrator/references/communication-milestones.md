# Communication Milestones

What to show Daniel at each pipeline milestone.

| Moment | Show |
|--------|------|
| **Inicio** | "Voy a crear un escape room de {temática}. Pipeline dual-LLM en 4 fases." |
| **Tras Explore** | Missing data → ask. OK → brief summary. Baseline → run Regression. |
| **Tras Regression** | Skip → "Juego nuevo, sin baseline." Concerns → show changes. Fail → escalate. |
| **Tras Conceive (dual)** | Title + tagline + premise (3 lines). "Concepto sintetizado de GLM-5.1 + GPT-5.5." |
| **Tras Design (dual)** | Puzzle summary + synthesis stats. "Diseño dual: {A} lógicas + {B} experienciales." |
| **Tras Difficulty Calibration** | Verdict + number of puzzles out of target. |
| **Tras Build** | Confirm build completed, announce playtest. |
| **Tras Playtest (dual)** | Verdict + X critical, Y warnings, Z divergences. "Playtest dual: 6 perfiles (3 analíticos + 3 experienciales)." |
| **Tras Verify** | Warnings/issues if any. |
| **Tras Judgment (dual)** | Each judge score + final verdict. "Juicio dual: GLM-5.1 ({score}) + GPT-5.5 ({score}) → {verdict}" |
| **Final** | "Juego completo en juegos/{juego}/. PDF generado." |
