# Cross-Game Comparison Rules

Compare new game against real completed games with playtest reports.

## What to Compare

1. **Difficulty curve**: Similar to successful games? 3→4→5→4 vs 2→7→3→8 = problem.
2. **Mechanic variety**: Real games use 5-6 mechanics. New game <3 = lacking variety.
3. **Estimated times**: Real games ~35min puzzles / ~15min transitions. New game should match ratio.
4. **Playtest scores**: Similar-themed games had problems with certain puzzle types → check new game doesn't repeat.
5. **Design patterns**: Must have ≥1 cooperative puzzle, max 2 same type, cumulative hilo conductor.

## Proven Design Patterns

- Minimum 1 cooperative puzzle (as in Legado Tinta Violeta, Legado de la Familia)
- Max 2 puzzles of same type (as in all existing games)
- Cumulative hilo conductor between puzzles (all project games)
