# Skill Success Metrics

## Working Well When

- **≥3 JSON puzzles** use it (after 1 month)
- **0 confusing queries** from agents about when to apply
- **Documented synergies** used in real games
- **Anti-patterns** prevent misuse
- **Decision time** <5 minutes for agent to choose

## Failing When

- Only 1 puzzle uses it after 3 months
- Agents ask "is this prueba-X or prueba-Y?"
- Defined variables unused in practice
- Consistently ignored in favor of others
