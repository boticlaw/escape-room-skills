# Compound Puzzle Examples

## Example 1: Naval Communication (Tower-Flags)
- **Skills**: `prueba-comunicacion-mensajes` + `prueba-busqueda-objetos` + `prueba-punteria-derribo`
- **Description**: Players knock down objects to reveal nautical flags. Dictionary hidden in a chest. Must decode message from water tower visible through window.
- **Why it works**: Combines physical (aim), exploration (search), mental (decoding)

## Example 2: Separated Spies (Morse-Binary)
- **Skills**: `prueba-comunicacion-mensajes` (×2) + `prueba-logica-secuencial`
- **Description**: Team A receives auditory morse, transmits to Team B. Team B has dictionary, decodes to binary. Binary activates panel in specific sequence.
- **Why it works**: Each team depends on the other, creating real cooperative tension.

## Example 3: Lighthouse of the End of the World
- **Skills**: `prueba-comunicacion-mensajes` + `prueba-panel-electrico` + `prueba-tablet-cooperativo`
- **Description**: Using tablets, one team controls ceiling lights (lighthouse) making morse. Other team sees via camera, decodes. Message indicates how to wire the escape panel.
- **Why it works**: Fuses digital technology with physical mechanics and classic code.
