# Flow Patterns

## Linear Simple (recommended for beginners)

```
Puzzle 1 → Code A → Puzzle 2 → Code B → Puzzle 3 → Victory
```

## Linear with Branching

```
Puzzle 1 → Code → [Puzzle 2a | Puzzle 2b] (parallel) → Puzzle 3 → Victory
```

## Accumulation

```
Puzzle 1 → Piece A
Puzzle 2 → Piece B
Puzzle 3 → Piece C
All pieces → Final Puzzle → Victory
```

## Competitive

```
Team A: Puzzle 1a → 2a → 3a → Victory?
Team B: Puzzle 1b → 2b → 3b → Victory?
(First to finish wins)
```
