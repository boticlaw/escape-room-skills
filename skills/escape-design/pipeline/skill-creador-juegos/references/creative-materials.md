# Creative Use of Available Materials

**Principle**: Materials create unique mechanics, not just decorative support.

| Material | Creative Use | Mechanic |
|----------|-------------|----------|
| **Tablet** | Password system | Code unlock, diary with timestamps, progressive gallery |
| **Microscope** | Hidden details | Watermarks, hidden layers, micro-text |
| **Locks** | Physical rewards | Derived codes (not guessable), tangible feedback |
| **Safes** | Next-step containers | Clues inside, keys, needed objects |
| **Flashlights** | Info reveal | Invisible ink, UV marks, dark elements |
| **Mirrors** | Alternative perspectives | Inverted text, hidden messages, reflections |
| **Clocks/timers** | Urgency & clues | Specific time as code, countdown as hint |

## Design Questions

1. What materials already available? → List all
2. What unique mechanic does each enable? → Not just "container"
3. How does it integrate in narrative? → Justify presence
4. Is it reusable? → Prioritize multi-game materials

## Simple Setup Rule

If a puzzle needs >15 min physical setup → redesign. Find alternatives with less infrastructure.
