# Validation Checklist

- [ ] Narrative connects all puzzles
- [ ] Variety of skills (max 2 repetitions)
- [ ] Adequate difficulty progression
- [ ] Coherent total estimated time
- [ ] Every puzzle has documented solution
- [ ] Every puzzle has ≥3 hint levels
- [ ] Physical materials are obtainable
- [ ] No circular dependencies
- [ ] Each puzzle reward leads to next
- [ ] Final puzzle integrates previous elements
- [ ] GM hints cover critical blocking points
- [ ] PDF generated successfully
