# Common Design Errors

| Error | Symptom | Fix |
|-------|---------|-----|
| Overload | 10+ puzzles in 45 min | Remove dispensable puzzles |
| No narrative | Disconnected puzzles | Create thread justifying each action |
| Inconsistent difficulty | Easy followed by impossible | Order by progression |
| Circular dependencies | A needs B that needs A | Diagram flow first |
